Dynamic Properties

A project I created to enable me prototype an MVC form with fields that vary depending on an answer given on the same form.

The project allows the user to define pet types (e.g. cat, dog, rabbit etc) which have a set of standard properties (e.g. name) and a number of dynamic properties that are specific to the pet type (for instance you might want to know how many times a day you walk a dog or how many liter trays you have for a cat).

Once some pet types have been defined (by default pet types dog & cat are included) the user can then add individual pets, changing the "Pet Type" drop down list will update the dynamic properties to reflect the pet type that has been selected.

I've used Knockout to facilitate the binding of the forms and to handle the refresh of properties in the pet entry screen, Knockout is all so used in the pet type entry screen to keep track of and add/edit properties, please note that the modal functionality used on the pet type edit screen is taken from http://aboutcode.net/2012/11/15/twitter-bootstrap-modals-and-knockoutjs.html and is not by own work.

The project runs in VS2013.

Usage (pet type editor);

  1. Make sure /Home/index is your start path
  2. Click "Pet Types" tab on the home screen (will be selected by default if you've just run the project)
  3. You can either edit one of the existing pet types (by clicking the appropriate "Edit" link in the list) or create a new one by clicking "Create New" link
  4. Enter a name for the pet type
  5. Click "Add property" to add a new property
  6. On the modal property window enter a property name (this will be what is displayed on the pet entry form)
  7. Select a property type from the drop down (this will affect validation on the pet entry form)
  8. Check whether the field is required or not (this will affect validation on the pet entry form, note boolean is unaffected by required)
  9. You can continue to add properties or if you have finished click "Save"
  10. It should be noted that by fixed property of "Name" is included in all pet types

Usage (pet editor);

  1. Make sure /Home/index is your start path
  2. Click "Pets" tab on the home screen
  3. You can either edit one of the existing pets (by default there won't be any on first run) by clicking the appropriate "Edit" link or create a new one by clicking "Create New" link
  4. Enter the name of the pet
  5. Choose a pet type
  6. Fill in the fields that appear
  7. Click "Save"
